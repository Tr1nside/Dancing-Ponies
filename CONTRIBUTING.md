# Contributing to Dancing Ponies

Спасибо за интерес к проекту! Вот несколько рекомендаций по участию.

## Как внести вклад

1. **Fork** репозиторий
2. Создайте **feature branch**: `git checkout -b feature/my-feature`
3. Внесите изменения
4. Убедитесь, что линтеры проходят:

   ```bash
   # Backend
   cd backend && ruff check . --fix && ruff format .

   # Frontend
   cd frontend && npm run check
   ```

5. Запустите тесты:

   ```bash
   cd backend && pytest
   ```

6. Создайте **Pull Request**

## Стандарты кода

### Backend (Python)

- **PEP 8** с максимальной длиной строки 120
- **Ruff** для линтинга и форматирования
- Типизация через type hints
- Docstrings для публичных функций

### Frontend (TypeScript/React)

- **Biome** как основной линтер/форматтер
- **ESLint** как дополнительная проверка
- Tab-индентация, двойные кавычки
- Компоненты — функциональные с хуками

## Сообщения о проблемах

При создании Issue укажите:

- Описание проблемы
- Шаги воспроизведения
- Ожидаемое vs фактическое поведение
- Окружение (ОС, браузер, версии)

## Лицензия

Внося изменения, вы соглашаетесь с тем, что ваш код будет распространяться под лицензией MIT.
